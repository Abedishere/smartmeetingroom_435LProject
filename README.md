# Smart Meeting Room Management System

Backend system for managing meeting rooms with Users and Rooms services.

## Services
- **Users Service** (Port 5001): User management with Admin role support
- **Rooms Service** (Port 5002): Room management with Facility Manager role support

## Tech Stack
- Python 3.13
- Flask
- PostgreSQL
- Docker
- JWT Authentication

## Setup
```bash
docker-compose up --build
```
